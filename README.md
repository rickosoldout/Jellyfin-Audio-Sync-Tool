# Jellyfin Audio Sync Tool

A CLI tool to search and download foreign audio tracks and inject them into your Jellyfin media library automatically.

## Setup

1. Clone the repo:

\`\`\`bash
git clone https://github.com/YOUR_USERNAME/jellyfin-audio-sync.git
cd jellyfin-audio-sync
\`\`\`

2. Create and activate Python virtual environment:

\`\`\`bash
python3 -m venv venv
source venv/bin/activate
\`\`\`

3. Install dependencies:

\`\`\`bash
pip install -r requirements.txt
\`\`\`

4. Copy `.env.example` to `.env` and edit with your config:

\`\`\`bash
cp .env.example .env
nano .env
\`\`\`

5. Run the tool:

\`\`\`bash
python jellyfin_media_tool.py
\`\`\`

## Usage

Choose from the menu to add new media or inject foreign audio.
